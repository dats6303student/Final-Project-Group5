train_test_split.py is used to create the file image_paths.xlsx.  The programs modelX_train.py where X is 1,2, or 3, are used to train the network.  The programs modelX_test.py where X is 1,2, or 3, are used to test the performance of the network.  All of these programs rely on the file image_paths.xlsx.  Also, parameters should be correctly specified when running each program.  The 1 programs are for 100X100 resizing, 2 is for 256x256 resizing, and 3 for 630x630 resizing.
